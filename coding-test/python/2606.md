# 🖥️ 백준 2606번 - 바이러 🔥🔥 Silver 3

## 📌 문제 설명
네트워크를 통해 전파되는 웜 바이러스가 있다. 1번 컴퓨터가 웜 바이러스에 걸렸을 때, 이 컴퓨터를 통해서 웜 바이러스에 감염되는 컴퓨터의 수를 구하시오.

## ⌨️ 입력 예시
```plaintext
7
6
1 2
2 3
1 5
5 2
5 6
4 7
```

## ⌨️ 출력 예시
```plaintext
4
```

## 🖥️ 풀이 코드
```python
cpu = int(input())
connect = int(input())

cpus = [set() for _ in range(cpu + 1)]
for _ in range(connect):
    pc1, pc2 = map(int, input().split())
    cpus[pc1].update([pc1, pc2])
    cpus[pc2].update([pc1, pc2])
    
visited = [False] * (cpu + 1)
def dfs(x):
    visited[x] = True
    count = 1
    for pc in cpus[x]:
        if not visited[pc]:
            count += dfs(pc)
    return count
    
print(dfs(1) - 1)
```

## 💡 풀이 포인트
- dfs 알고리즘의 활용
- 집합을 통한 중복 요소 관리
